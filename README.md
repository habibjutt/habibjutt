## About Me

I'm a **Full-Stack Developer** based in **Dubai, UAE** - building modern web apps, SaaS products, and developer tools.

- Currently working at **Gulf Estates** - building real estate tech
- Building **open-source** tools like [Layout Generator](https://grid-generator.devtrone.com/) and [SaaS Boilerplates](https://github.com/habibjutt/nextjs_boilerplate)
- Experienced across the full stack - from **React/Next.js** frontends to **Node.js/Laravel** backends
- Comfortable with **Docker, Kubernetes**, and CI/CD pipelines
- Always learning - currently diving deeper into **AI integrations** and **DevOps automation**
